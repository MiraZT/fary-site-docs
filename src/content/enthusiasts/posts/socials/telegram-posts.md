# Telegram-посты

## 📝 О документе

Этот документ описывает **специфические требования и шаблоны для публикаций в Telegram-канале** проекта.  
Общие принципы написания текстов, форматирования, тона и структуры статей изложены в **[общем руководстве по постам](../general.md)** — перед подготовкой материалов для Telegram обязательно ознакомьтесь с ним.

Все перечисленные ниже элементы являются обязательными для сохранения единого стиля канала.

## 🏗️ Специфическая структура Telegram-постов

### Базовый шаблон

```markdown
❔ Письмо поста от [дата]. Редакция: [статус].

✍️ [дисклеймер]

[приветствие]

[основной текст]

🤜 Лучше вместе! 🤛
```

### Расширенный шаблон с секциями (для больших постов)

```markdown
❔ Письмо поста от [дата]. Редакция: [статус].

✍️ [дисклеймер]

[приветствие]

⫷ ИГРОКАМ ⫸

[контент для игроков]

⫷ КОМАНДЕ ⫸

[контент для команды]

🤜 Лучше вместе! 🤛
```

## 💡 Дополнительные указания для Telegram

1. **Смайлы-разделители** – используйте эмодзи для визуального отделения логических блоков _(например, ⫷ … ⫸ или ❔, ✍️)_.
2. **Жирное выделение** – ключевые фразы и предупреждения обязательно выделяйте **жирным**.
3. **Адаптация под мобильные экраны** – абзацы должны быть короткими _(3–4 строки)_, чтобы текст не «сливался» на узких экранах.

Все остальные требования _(типографика, использование «ёлочек», обращение на «вы» и пр.)_ полностью соответствуют **[общим правилам оформления постов](../general.md)**.
